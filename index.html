Physics Projects

These have been done either as personal projects or mandated to me by my teachers. Still love em.

Nonlinear Heat Transfer

This is a simulation that I made to see if you could do something like a red-black scheme with einstien solids to model heat transfer in nano structures.

Fast 2D poisson Solver

An example program thats used to show various iterative techniques to solving Poisson's equation.

Projectile Motion

A Project to just show how fast you can get by using c and numerical methods. I use a combination of fast math and an optomized newton's method to solve the projectile motion of an object experiencing air resistance. I got this thing to converge, on average, in 12 ns.

Coupled Oscillators

A python script that solves a set of coupled ODEs concurrently. I used this to analyse the various phasespaces of the system and characterize it. It's pretty cool, and has a set of conditions where there are multiple attractors with stable orbits.
